# 가상 공연 플랫폼

## 개요
* 플랫폼 이름
  * STAGE ON
* 프로젝트 기획 의도 
  * 실시간으로 인터랙션하는 메타버스 버츄얼 공연 플랫폼
* 프로젝트 목표
  * 퍼포머와 관객 간의 실시간 상호작용 및 몰입감 있는 공연 제공
* 제작 인원
  * 총 7명 : 기획자 1, 개발자 3, TA 1, BE 1, AI 1
* 프로젝트 기간 
  * 24.09.30-11.20(약 2달)
* 시연 영상 : [Click here to try](https://url.kr/7r6jry)

## R(Role) & R(Responsibility) 

### 유승호(기획)
 * 역할
   * 팀장 및 PM
 * 담당 업무
   * 기획서 제작
   * 화면 구성, 플로우 차트
   * PPT 제작

### 김민경(XR)
  * 역할
    * 팀원
  * 담당 업무
    * 공연 예약
    * 세션 관리
    * UI 연결 및 구현
    * 관객 간의 인터렉션
      
### 조준혁(XR)
  * 역할
    * 팀원
  * 담당 업무
    * 무대 제작
    * 버튜버 모델링 적용
    * UI 연결 및 구현
    * 버튜버 연결
      
### 한승우(XR)
  * 역할
    * 팀원
  * 담당 업무
    * 관객
    * 버튜버와 관객 간 인터렉션
    * UI 연결 및 구현
    * 사운드 관련
      
### 전상우(TA)
  * 역할
    * 팀원
  * 담당 업무
    * 모션 캡처
    * 커스터마이징 레벨 모델링
    * 관객 모델링
    * UI 디자인

### 유예나(AI)
  * 역할
    * 부팀장
  * 담당 업무
    * 프롬폼트 기반 AI 이미지 생성
    * 실시간 댓글 번역
    * API 개발 및 구현
   
### 윤여원(BE)
  * 역할
    * 팀원
  * 담당 업무
    * 홈페이지 제작 및 개발
    * 회원정보 관리
    * 공연정보 관리
    * 티켓 콜렉션 기능

**브렌치명 컨벤션**
- 브렌치 명 : <자기이니셜>_<기능>

**브렌치명 컨벤션 예**

- HSW_PlayerMove
- CJH_BasicSetting

<br/> 

**커밋 컨벤션**

- [유형] <내용> #이슈번호
- Add : 파일이 추가된 경우
- Feat : 새로운 기능을 추가할 경우
- Edit : 코드나 에디터 등 수정이 이루어진 경우
- Fix : 버그를 고친 경우
- Rename : 파일 혹은 폴더명을 수정하거나 경로 수정을 한 경우
- Remove : 파일을 삭제하는 경우
- !HOTFIX : 치명적인 버그를 고친 경우
- Style : Norm 규정, 세미콜론 누락, 코드수정이 없는 경우
- Refactor : 리팩토링
- Comment : 주석 추가 및 변경
- Docs : 문서를 수정한 경우
- Test : 테스트 추가


**커밋 컨벤션 예**

- [Feat] tokenizer 구현
- [Fix] tokenizer 문자열 파싱 버그 수정
- [Remove] test.txt 임시파일 삭제
- [Rename] test.txt → operator.txt

<br/> 

**코드 컨벤션**

- 블루프린트 이름 규칙: BP_블루프린트이름_이니셜
- 클래스 이름 규칙: 클래스이름_이니셜
- 전역변수: 파스칼 표기법 
- 지역변수: 카멜 표기법 
- 불 변수: b로 시작
  

** 코드 컨벤션 예**

블루프린트, 클래스 이름
- BP_TestCharacter_HSW
- NetGameInstance_CJH
- Player_KMK

변수 이름
- 전역변수: ShootingPlayer
- 지역변수: gameInstance
- 불변수 : bIsPlaying
